Initialise the kernel's input, output and border mode.

Parameters
[in]input1An input tensor. Data types supported: U8/S16/F16/F32

[out]outputThe output tensor. Data types supported: U8 (Only if both inputs are U8), S16/F16 (only if input1 is F16), F32 (only if both inputs are F32).
[in]scaleScale to apply after multiplication. Scale must be positive and its value must be either 1/255 or 1/2^n where n is between 0 and 15.
[in]overflow_policyOverflow policy.
[in]rounding_policyRounding policy.
